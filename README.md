# AquaStep

AquaStepは、まちの中にある水たまりの場所や危険度を確認できるWebアプリです。地図上の水たまりを選ぶと、写真や深さ、大きさ、靴と比べたときの目安、転びやすさや夜間の見えにくさなどを確認できます。

雨上がりの道を歩くときに、足元の危険を事前に知るためのアプリです。

## 主な機能

- 地図で水たまりの場所を確認できます。
- 水たまりを選ぶと、写真・深さ・大きさ・観測日などを確認できます。
- 靴と水たまりの大きさを比べて、歩くときの注意度をイメージできます。
- すべりやすさ、転びやすさ、夜間の見えにくさをリスクとして表示します。
- 時間帯を変えて、明るい時間と暗い時間での見え方を確認できます。
- 水たまりの注意情報を音声で聞くことができます。
- 写真、場所、深さ、大きさ、観測日を入力して水たまり情報を登録できます。
  - 現在地取得に対応しています。

## 使い方

公開中のアプリはこちらから利用できます。

[https://aqua-step-nine.vercel.app](https://aqua-step-nine.vercel.app)

1. アプリを開きます。
2. 地図上の水たまりマーカーを選びます。
3. 画面下の詳細表示で、水たまりの写真や危険度を確認します。
4. `NAVI` を押すと、選択中の水たまりについて音声で注意情報を確認できます。
5. `ADD` を押すと、水たまり情報を登録できます。

## ローカルで動かす場合

自分のPCで動かす場合は、以下の手順で起動できます。

### 1. 依存関係をインストール

```bash
npm install
```

### 2. 環境変数を設定

データ保存先としてSupabaseを利用しています。プロジェクトルートに `.env.local` を作成し、以下を設定します。

```bash
SUPABASE_URL="supabase_project_url"
SUPABASE_SERVICE_ROLE_KEY="supabase_service_role_key"
```

写真を保存するため、Supabase Storageに `puddle-photos` バケットが必要です。設定用SQLは `supabase/setup.sql` にあります。

### 3. 開発サーバーを起動

```bash
npm run dev
```

ブラウザで `http://localhost:3000` を開きます。

### 4. 本番用ビルド

```bash
npm run build
npm run start
```

## 使用したデータ・技術、ライブラリ

### データ

- 水たまりの位置、深さ、大きさ、観測日などの情報
- 投稿された水たまりの写真
- 地図上で扱いやすいGeoJSON形式の位置データ

### 技術・ライブラリ

- Next.js: Webアプリの画面とAPIを作るために使用
- React: 画面の表示や操作部分を作るために使用
- TypeScript: コードを安全に書くために使用
- Leaflet / React Leaflet: 地図表示のために使用
- Supabase: 水たまり情報と写真の保存に使用
- Lucide React: 画面内のアイコン表示に使用
